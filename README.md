# LinuxForge

LinuxForge is an interactive Linux command learning hub with a built-in terminal simulator, structured command references, and progressive learning paths from beginner to advanced.

It is designed as a single-file, fast-loading web app you can host anywhere.

## Project Description (GitHub Short Description)

Interactive Linux command reference and terminal simulator built in pure HTML/CSS/JavaScript.

## Highlights

- 60+ Linux command guides across multiple categories
- Beginner, intermediate, and advanced learning tracks
- Search and level filtering in the sidebar
- Rich command docs: syntax, flags, examples, tips, and warnings
- Built-in terminal simulator with command history and autocomplete
- Virtual file system simulation for hands-on practice
- Clean, modern UI with responsive behavior

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no framework)

## Repository Structure

- linux-hub.html - Complete application (UI, data, logic)

## Run Locally

1. Clone the repository.
2. Open linux-hub.html in your browser.

Optional local server:

- Python:
  - python3 -m http.server 8000
  - Open http://localhost:8000

## Customization

You can quickly customize:

- Brand name and terminal hostname strings
- Command database (add/edit command entries)
- Colors and typography in CSS variables
- Virtual file system structure in the JavaScript VFS object

## Deployment

Because this is a static app, you can deploy it on:

- GitHub Pages
- Netlify
- Vercel (static)
- Any shared hosting or VPS
